Definition: Discrete Random Variable

A discrete random variable is a variable that can take on a countable number of distinct values. Each possible value has an associated probability.

Definition: Probability Distribution Function (PDF)

For a discrete random variable X, the probability distribution function P(X = x) gives the probability that X takes the value x. It must satisfy:

  • 0 â‰¤ P(X = x) â‰¤ 1 for all x
  • âˆ‘ P(X = x) = 1 (sum over all possible values)

Key Properties of Discrete Probability Distributions

  1. Non-negativity: All probabilities are non-negative
  2. Normalization: The sum of all probabilities equals 1
  3. Countable outcomes: The sample space is countable
  4. Discrete values: No values between defined points

Definition: Binomial Distribution

A binomial distribution models the number of successes in a fixed number of independent trials, where each trial has the same probability of success.

Notation: X ~ Binomial(n, p) or X ~ B(n, p)

Parameters:

  • n: Number of trials (fixed)
  • p: Probability of success in each trial (constant)
  • X: Number of successes (0, 1, 2, ..., n)

Binomial Probability Formula

P(X = k) = C(n,k) Ã— p^k Ã— (1-p)^(n-k)

Where:

  • C(n,k) = n!/(k!(n-k)!) is the binomial coefficient
  • k = number of successes (0 â‰¤ k â‰¤ n)
  • p = probability of success
  • (1-p) = probability of failure

Binomial Distribution: Expected Value and Variance

E(X) = Î¼ = np
Var(X) = ÏƒÂ² = np(1-p)
Ïƒ = âˆš(np(1-p))

Conditions for Binomial Distribution

  1. Fixed number of trials (n)
  2. Each trial has two outcomes: success or failure
  3. Constant probability: P(success) = p for all trials
  4. Independence: Trials are independent of each other
